Tôi sửa cái nì hihi

  1. Lấy project từ git về máy
    2.             B1. Vào trang github
            B2. Chọn project cần lấy về máy
            B3. Copy URL của project
            B4. Mở vị trí đặt project
            B5. Nhấn chuột phải rồi chọn git bash here
            B6. Trên màn hình console, gõ lệnh: 
                git clone <URL của project>
            B7. Tại vị trí đó sẽ xuất hiện project
  2. Tải project từ máy lên git(project này đã có liên kết với repos từ trước)
    3.                 B1: Nhấn chọn project
                B2: Trong thư mục project, nhấn chuột phải rồi chọn git bash here
                B3: Gõ lệnh: "git add ." để thêm các sửa đổi vừa thực hiện vào hàng đợi
                B4: Gõ lệnh: "git commit -m 'Mô tả chỉnh sửa'" để xác nhận các sửa đổi
                    và thêm mô tả cho sửa đổi đó
                B5: Gõ lệnh: "git push" để đẩy project đã sửa đổi lên github
  3. Tải project từ máy lên git(project này chưa có liên kết với repos từ trước)
    4.                 Vì project chưa liên kết với kho chứa nào trên github nên chúng ta phải tạo mới 1 repos
                hoặc chọn 1 repos có sẵn, copy URL của repos đó để chuẩn bị cho việc liên kết
                với project trên máy tính
                B1: Vào github, tạo mới 1 repos hoặc chọn 1 repos có sẵn
                B2: Copy URL của repos vừa chọn
                B3: Vào thư mục project trên máy tính
                B4: Nhấn chuột phải rồi chọn git bash here
                B5: Trong màn hình console, gõ lệnh:
                    "git init -b main"
                    Nếu ta ghi "git init" thì project của chúng ta sẽ được quản lí bởi nhánh(branch) master
                    chứ không phải nhánh main, vậy nên phải thêm "-b main" để project của chúng ta chuyển vào nhánh
                    main trong git
                B6: Sau khi lệnh này được thực thi, project trên máy tính sẽ đc quản lí bởi git và có thể
                    liên kết với 1 repos trên git bằng câu lệnh:
                    "git remote add origin <URL của repos>"
                B7: Gõ lệnh:
                    "git add ."
                    "git commit -m <Mô tả chỉnh sửa>"
                B8: Liên kết project trong máy tính với repos trên git bằng lệnh:
                    "git remote add origin <URL của repos>"
                B9: Gõ lệnh:
                    "git push" 
                    Lưu ý:
                    Khi gõ lệnh "git push" thì git bash sẽ yêu cầu bạn chỉ rõ push project lên nhánh nào
                    Vậy nên ta phải chỉ định rõ project của chúng ta thuộc nhánh nào
                    Giả sử project của chúng ta thuộc nhánh main, ta gõ lệnh:
                    "git push --set-upstream origin main"
                B10: Nếu git báo lỗi sau:
                
                Đây là lỗi từ phía git
                Chúng ta có 2 cách để khắc phục:
                Cách 1: Gõ lệnh:
                    "git pull --rebase origin main"
                    hoặc
                    "git pull origin main"
                    Lệnh này sẽ áp dụng các sửa đổi trên repos git lên project local
                    Việc sửa đổi này chỉ được áp dụng ở nhánh main
                    Sau đó ta gõ lệnh:
                    "git push origin main"
                Cách 2: Ép buộc push
                    "git push -f origin main"
                    -f là viết tắt của force(ép buộc)
                    push ép buộc
                    Tức là ta sẽ đẩy project local lên repos git 1 cách ép buộc
                    Tại vị trí nhánh main
  4. Cập nhật những sửa đổi của project trên git vào project trên máy
    5.                 Gõ lệnh: "git pull"
  5. Tạo ra các nhánh con để không ảnh hưởng đến nhánh chính
    6.                 1. Để xem các nhánh đang tồn tại, gõ lệnh:
                "git branch"
                2. Nhánh có dấu * ở đầu là nhánh mình đang làm việc
                3. Để thêm nhánh mới, ta gõ lệnh:
                "git branch <Tên nhánh mới>"
                4. Để chuyển sang làm việc trên 1 nhánh khác, gõ lệnh:
                "git checkout <Tên nhánh cần chuyển đến>"
